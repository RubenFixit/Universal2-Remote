# IR blaster Assembly

![](images/ir_blaster_open.jpg)


Simply connect a micro USB breakout board to the Sparkfun WiFi IR blaster as shown above.

```
IR Blaster   USB Micro
_____         ____
     |       |
     |    /--| 5V
     |   /   |
  Vin|--/    |
  GND|-------|GND
_____|       |____

```

Then solder a 6 pin female header to the bottom of the IR blaster.
